# Todo List App

This is a simple Todo List web application built using HTML, CSS, and JavaScript.  
It helps users add, complete, edit, and delete daily tasks in a clean and easy way.

---

## Features

- Add new tasks  
- Mark tasks as completed  
- Edit existing tasks  
- Delete tasks  
- Simple and modern glossy UI  
- Fully responsive design (works on mobile and desktop)

---

## Tech Stack

- HTML  
- CSS  
- JavaScript (DOM Manipulation)

---

## How to Run

1. Download or clone this project  
2. Open the `index.html` file in your browser  
3. Start adding your tasks

---

## Project Purpose

This project was built for practice to improve frontend development skills and understand JavaScript DOM manipulation better.

---

## Note

This is a beginner-friendly project focused on basic CRUD operations in JavaScript.
